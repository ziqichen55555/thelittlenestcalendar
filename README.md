# 房间预订日历 🏠

一个简单易用的房间预订管理系统，用于和朋友查看房间的预定情况。

## 功能特点

- 📅 **日历视图** - 直观的月历显示，清晰展示预订日期
- ➕ **创建预订** - 点击日期或使用"新建预订"按钮创建预订
- ✏️ **编辑预订** - 随时修改预订信息（日期、人数、备注）
- 🗑️ **删除预订** - 轻松删除不需要的预订
- 👥 **人数显示** - 每个预订显示入住人数
- 📝 **备注功能** - 为每个预订添加备注信息
- 💾 **本地存储** - 数据保存在浏览器本地，无需服务器

## 使用方法

### 安装依赖

```bash
npm install
```

### 启动开发服务器

```bash
npm run dev
```

应用将在 `http://localhost:5173` 启动

### 构建生产版本

```bash
npm run build
```

## 使用说明

1. **查看日历** - 主界面显示当前月份的日历，已预订的日期会以紫色高亮显示
2. **创建预订** - 点击任意日期或点击"新建预订"按钮，填写：
   - 开始日期和结束日期
   - 入住人数
   - 备注信息（可选）
3. **查看预订** - 右侧边栏显示所有预订列表，包括日期范围和详细信息
4. **编辑预订** - 点击预订列表中的"编辑"按钮修改预订信息
5. **删除预订** - 点击"删除"按钮移除预订

## 技术栈

- React 18
- TypeScript
- Vite
- CSS3 (渐变、动画效果)

## 数据存储

所有预订数据保存在浏览器的 localStorage 中，刷新页面后数据不会丢失。
