# 📍 如何找到 GitHub Token 创建页面

## 🎯 最简单的方法：直接链接

**直接点击这个链接：**
👉 **https://github.com/settings/tokens/new?type=classic**

## 📋 如果直接链接不行，按以下步骤：

### 步骤 1：进入设置
1. 登录 GitHub
2. 点击右上角你的**头像**
3. 点击 **"Settings"**（设置）

### 步骤 2：找到 Developer settings
1. 在左侧菜单中，**向下滚动**
2. 找到最底部的 **"Developer settings"**（开发者设置）
3. 点击它

### 步骤 3：进入 Personal access tokens
1. 在左侧菜单中，点击 **"Personal access tokens"**（个人访问令牌）
2. 点击 **"Tokens (classic)"**（经典令牌）

### 步骤 4：创建新 Token
1. 点击 **"Generate new token"**（生成新令牌）
2. 点击 **"Generate new token (classic)"**（生成经典令牌）

## ✏️ 填写 Token 信息

1. **Note（备注）**：输入 `thelittlenestcalendar`
2. **Expiration（过期时间）**：选择 **"No expiration"**（永不过期）
3. **Select scopes（选择权限）**：
   - 向下滚动找到 **"gist"**
   - **只勾选 `gist`**（其他都不要勾选）
4. 滚动到页面最底部
5. 点击 **"Generate token"**（生成令牌）
6. **立即复制 Token**（格式类似：`ghp_xxxxxxxxxxxxxxxxxxxxxxxxxxxxxxxxxxxx`）

## ⚠️ 重要提示

- Token **只显示一次**，离开页面就看不到了
- 如果忘记复制，需要删除旧 Token 重新创建
